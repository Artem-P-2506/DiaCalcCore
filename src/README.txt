============================== Документація до ядра програми "DiaCalc" ==============================

---------- Опис програми: ----------
Ця програма призначена для розрахунку дози інсуліну для людей із цукровим діабетом.

Основні функції включають:
1. Отримання харчової інформації продуктів через API Edamam.
2. Використання цих даних для розрахунку інсулінових доз на основі індивідуальних параметрів користувача.
3. Гнучка система обчислень, що враховує рівень цукру в крові, прийом їжі та інші чинники.

---------- Архітектура програми: ----------
1. Клас DiaCalcCoreTest (Точка входу):
   * Взаємодія з користувачем через консоль.
   * Отримання введення користувача (дані про рівень цукру, харчування тощо).
   * Запит до API для отримання харчових даних.
   * Виклик методів для розрахунку доз інсуліну.
   * Виведення результатів розрахунків.

2. Клас EdamamAPI:
   * Виконує HTTP-запити до сервісу Edamam для отримання харчових даних.
   * Обробляє відповіді API у форматі JSON.
   * Витягує дані про білки, жири та вуглеводи для заданого продукту.
   * Обробляє помилки, пов’язані з API-запитами.

3. Клас InsulinCalculator:
   * Розраховує коригуючу дозу інсуліну залежно від поточного рівня цукру в крові.
   * Обчислює дозу інсуліну для вуглеводів із врахуванням типу прийому їжі (сніданок, обід, вечеря).
   * Визначає загальну дозу інсуліну на основі коригуючої дози та дози на вуглеводи.

4. Клас NutritionalInfo:
   * Зберігає дані про харчовий склад продукту (жири, білки, вуглеводи).
   * Надає методи доступу до цих даних.
   * Використовується для передачі даних між API та розрахунками.

5. Клас ConfigAPI:
   * Завантажує конфігураційні дані (APP_ID та APP_KEY) з файлу `configAPI.properties`.
   * Забезпечує безпечний доступ до ключів API для роботи з Edamam.

6. Файл (який не вноситься на GitHUB) з конфігуряціями (ключами) для API (EdamamAPI).
    Приклад файлу " configAPI.properties ":
API_id=Ваш_APP_ID
API_key=Ваш_APP_KEY

---------- Використання програми: ----------
1. Переконайтеся, що файл `configAPI.properties` знаходиться у вказаному шляху та містить правильні APP_ID і APP_KEY.
2. Запустіть програму DiaCalcCoreTest.
3. Введіть параметри, що запитує програма (рівень цукру, тип їжі, продукт тощо).
4. Отримайте розраховану дозу інсуліну у відповідь.


---------- Примітка: ----------
Програма є прототипом і призначена для демонстрації роботи. Для реального використання необхідно проконсультуватися з лікарем.
